# DESK — Workspace Store

## 🚀 Live Demo

**[Click here to try the live demo]()**

This is my **first JavaScript project**, developed to practice core JS concepts like classes, DOM manipulation and dynamic rendering.

## Features

- Product browsing with category filtering (All, Electronics, Stationery, Furniture)
- Add items to cart with real-time button updates 
- Smooth sliding cart drawer with overlay
- Live subtotal, total and cart badge
- Automatic unique category tags
- Empty cart state and toast notifications
- Fully responsive design

## Key Functions & Technical Implementation

- **Product Class** — Defines product structure and `getFormattedPrice()` method.
- **Cart Class** — Handles `addItem()`, `removeItem()`, `getTotal()`, `getItemCount()`, and `clear()`.
- **Rendering** — `renderProducts()` and `renderCart()` keep the UI in sync efficiently.
- Uses modern JavaScript: classes, `map()`, `filter()`, `reduce()`, `find()`, `Set`, and template literals.

## Technologies Used

- HTML5, CSS3, Vanilla JavaScript (ES6+)
